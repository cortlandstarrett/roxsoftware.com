General Instructions

  1. Build one of the Lego robots using the step by step assembly instructions provided below.
  2. Download one of the sample UML models and import into BridgePoint.
  3. Modify the UML model to include your own design ideas.
  4. Upload and translate the UML model using the MC-3020 UML Model Translation Server.
  5. Download the generated code/link from the server.
  6. Download into your Lego robot.
  7. Run it.
  8. Enter or host a contest!
